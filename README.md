# UPS Clone

Project from Sonny Sangha's YouTube channel.

---

## Setup

Install node_modules

```
npm install
```

Start Expo

```
expo start
```

Run Tailwind in development mode:

```
npm run dev:tailwind
```

Run StepZen

```
cd stepzen
stepzen start
```
